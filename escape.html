외부 브라우저 탈출 테스트 🚪 v5 · 실제 적용용

카톡으로 이 링크를 본인에게 보내 → 카톡 안에서 열고 → 탈출 버튼 → 외부 브라우저에서 리뷰 링크가 정상 동작하는지 확인. iOS·AOS 둘 다 테스트하세요.

현재 환경

① 탈출 (카톡 → 기본 브라우저)

실제 적용 땐 보통 배너+버튼(수동)이 안전. 자동은 유저 제스처 없이 막힐 수 있어 비교용으로만.

② 탈출 후 리뷰 링크 동작 확인 (생짜 window.open)

외부 브라우저에선 이 링크들이 앱(설치시)으로 알아서 뜹니다. 카톡 안에서 누르면 갇히는 것과 비교해보세요.
